# 航运知识文档目录

请将航运相关的知识文档放入此目录，支持的格式：
- PDF 文件（.pdf）
- 文本文件（.txt）

## 建议收集的文档类型

1. **航海学教材**
   - 航海学基础
   - 船舶操纵与避碰
   - 航海气象与海洋学

2. **轮机工程**
   - 船舶柴油机
   - 船舶辅机
   - 轮机维护与管理

3. **电子电气**
   - 船舶电气设备
   - 船舶自动化
   - 电子导航设备

4. **GMDSS 通信**
   - 全球海上遇险与安全系统
   - 无线电通信规则

5. **船员培训资料**
   - 船员适任考试题库
   - 海事法规
   - 安全管理

## 使用方法

1. 将文档放入此目录
2. 运行 `python RAG/rag_builder.py` 构建知识库
3. 知识库会自动保存到 `RAG/vectorstore/` 目录
